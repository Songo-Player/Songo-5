## Forking Songo#5
If you want to fork Songo#5 feel free. You can use the Songodot runtime and I dont care if I am credited at all. Though I would appreciate if fixes get merged back into the runtime if you find any bugs. 
Regarding forking the Songo#5 app, I would appreciate some sort of crediting. I would also *prefer* that you dont use "Songo#5" in the title of a forked variation of the app unless your fork is primarily a port and doesn't change/add to core functionality. I don't mind if you use "Songo" in a forked repo title.

- "Songo#5 Bega edition" == BAD
- "Songo Redux" == OK

The above represents my preference and ultimately isn't something I can or will try to enforce.
